# transitions

- 이동/변경 을 멋지게 보여주는 효과.

- 트랜지션을 적용하려면, 무엇을 변경할지 적어주면 됨.

- ease-in-out;

  > 스르륵나타나고 스르륵 사라지는 효과

- transition:background-color 5s ease-in-out;

  > 5초안에 스르륵 효과와 함께 바뀜

- 트랜지션은 이렇게 어떤 state가 바뀔때 적용이 된다. 

- state

  > hover, active, focus, visited등이있다.

- ##### transformation

  > html 문서의 element들을 변경, 모습이 변하는 효과를 뜻한다.
